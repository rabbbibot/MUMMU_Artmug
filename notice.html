커미션 및 이용 안내

  • 추가 수정 추가 수정은 마감일(작업물 전달일)로부터 3달 내 가능합니다.
    • 수정은 최대 2회까지 가능합니다.
    • 리깅 과정에서 파츠 수정 및 일러스트가 들어갈 경우 작업일이 늘어 날 수 있습니다.
    • 작업자의 실수로 생긴 오류나 작업 과정에서 파악하지 못한 수정사항이 있을 경우 3달 내 연락주셔야 수정이 가능합니다.
  • 모델 설치 경로 아래 폴더에 압축파일을 풀어서 넣어주시면 됩니다.
    C:\Program Files (x86)\Steam\steamapps\common\VTube Studio\VTube Studio_Data\StreamingAssets\Live2DModels

    내 컴퓨터 상단 주소창에 위 경로를 붙여넣기 하시면 됩니다.

    • 혹은 Steam → Library
    • Steam에서 폴더 열기
      1. VTube Studio 실행
      2. 모델 (화면 좌측 메뉴 첫번째)
      3. 원하는 모델 가져오기
      4. 폴더 열기
      5. 해당 폴더에 압축 풀기
  • 모델 셋팅 보내드린 모델은 작업자(MUMMU)에게 맞춰 셋팅되어 있기 때문에 반드시 본인에게 맞는 셋팅으로 변경해주셔야 합니다.
    • VTube Studio / OBS 최적화 셋팅으로 도와드리고 있습니다. 디스코드 화면공유로 진행하며 @mummuuyo 로 연락 부탁드립니다.
  • 모델 양도 / 판매 모델 양도, 판매를 원하실 경우 메일(mummuuyo@gmail.com)로 사전에 연락 부탁드립니다.
    • 판매 완료 후 구매자와 반드시 연결을 해주셔야 합니다.
